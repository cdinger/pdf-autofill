PDF Autofill

PDF Autofill is a lightweight web service that automatically fills PDF form fields with known data.

Usage

Using the autofill service is as simple as changing your existing PDF links to go through pdf-autofill. For example, if you're currently linking like:

Complete <a href="some_gnarly_form.pdf">this painful form</a>.

to:

Complete <a href="https://yourorg.com/pdf-autofill/autofill?some_gnarly_form.pdf">this painful form</a>.

Available fields

Field name Description
autofill/fake_field This is a placeholder for the template.
